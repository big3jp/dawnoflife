# 🌟 Cute 3D Action Puzzle: Star Collector 🌟

## Overview
Welcome to **Star Collector**! A beautiful, bouncy, and cute 3D action puzzle game that runs completely in your web browser. 

No installation, no plugins, and no complicated setup. It's built beautifully with **Three.js** and HTML5.

## How to Play
1. **Open `index.html`** in any modern web browser (Google Chrome, Microsoft Edge, Firefox, etc.). Just double-click the file!
2. Use **WASD** or **Arrow Keys** to roll around the pastel island.
3. Collect all the spinning **Gold Stars (☆)**.
4. Reach the **Orange Exit Tile** to progress!
5. Don't fall off the island! 

## Features
- **Zero Dependencies**: Uses an import map to fetch Three.js from a CDN. No Node.js or `npm install` needed!
- **Squash & Stretch Animations**: Feel the impact of every jump with custom procedural tweening logic.
- **Beautiful Pastel Aesthetics**: Handcrafted color palette to ensure the most "kawaii" visuals. Soft shadows and toon shading natively via WebGL.

*Have fun playing!*
